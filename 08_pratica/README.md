### Scripts da pratica

Neste diretório, está contido o script `script_pratica_08.sh` que contém todos os comandos utilizados para executar tarefas propostas nesta [prática 08](pratica_08_shell.pdf).

Para executar o script, basta executar uma das opções abaixo:

```
../08_pratica$ sudo chmod +777 script_pratica_08.sh
../08_pratica$ sudo ./script_pratica_08.sh -argumento_1 -.. -..

ou

../08_pratica$ sudo bash script_pratica_08.sh -argumento_1 -.. -..
```
